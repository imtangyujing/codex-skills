# Whiteboard Rules

绘制或更新Feishu画板版SOP时使用这些规则。

- Flowchart steps should carry their own embedded text. Do not place an extra text box inside a process node.
- Separate text boxes are fine for title, section headers, notes, legends, and timeline annotations.
- Keep each step node to one job:输入、动作、输出状态.
- Put status names in the node footer or the same embedded text block.
- Highlight human checkpoints with a distinct border or color, then keep the wording short.
- Add a small note near妙记自动化:可靠触发来自命名规范；链接只作备份入口。
- For schedule notes, use`12:00` and`18:00`.
